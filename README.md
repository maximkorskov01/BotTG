# Квиз-бот

## Команды бота

* /start - запуск квиза
* /help - помощь
* /quiz - начало игры

## Описание

Бот проводит интерактивный квиз с 15 вопросами. После прохождения показывает результат правильных ответов и сохраняет их.

## Пример использования

1. Запустите бота командой /start
2. Начните игру /quiz 
3. Отвечайте на вопросы
